# Politique de Sécurité (Security Policy)

## Versions Supportées
Nous appliquons activement des correctifs de sécurité uniquement sur la branche principale du projet de fin de cycle.

| Version | Supportée |
| ------- | --------- |
| 1.0.x   | ✅ Oui    |

## Signaler une Vulnérabilité
Si vous découvrez une faille de sécurité ou une vulnérabilité sur l'application, merci de ne pas ouvrir de ticket public (Issue). 

Veuillez contacter l'équipe de développement directement par e-mail ou via nos canaux de communication internes pour nous permettre de corriger le problème rapidement.
